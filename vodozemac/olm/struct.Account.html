pub fn pickle_to_json_string(&self) -> AccountPickledJSON

Pickle the Olm account and serialize it to a JSON string.

The string is wrapped in AccountPickledJSON which can be derefed to access the content as a string slice. The string will zeroize itself when it drops to prevent secrets contained inside from lingering in memory.
